Function

A binary relation R from A to B is said to be a function if ∀ a ∈ A, ∃ a unique element b ∈ B such that ( a, b ) ∈ R. [ ( a, b ) ∈ R can be written as R (a) = b ] Set A is called domain & B is called range of the function R.

  1.  [Commutativity]: x ⋁ y = y ⋁ x
              & x ⋀ y = y ⋀ x
  2.  [Distributive]: x ⋁ (y ⋀ z) = (x ⋁ y) ⋀ (x ⋁ z)
           & x ⋀ (y ⋁ z) = (x ⋀ y) ⋁ (x ⋀ z)
  3.  [Identity element]: There exists element 0, 1 ∈ S such that
                x ⋁ 0 = x
               & x ⋀ 1 = x
  4.  [Inverse]: x ⋁ ¬ x = 1
        & x ⋀ ¬ x = 0

Example

  1. Let S be a non empty set then P(s) is a Boolean algebra with ⋁ = ∪, ⋀ = ∩, ¬ A = Ac, 0 = p and 1 = S. This is called the power set Boolean algebera. So we have Boolean algebra of finite size as well as uncountable size.
  2. Take D(30) = {n ∈ N : n|30} with a ⋁ b = LCM(a, b)                 a ⋀ b = GCD(a, b)
                &   ¬ a = 30/a
    It is a Boolean algebra with 0 = 1 & 1 = 30
