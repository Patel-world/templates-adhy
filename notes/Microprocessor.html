                           MICROPROCESSOR

The microprocessor is a programmable integrated device that has computing and decision-making capability similar to that of CPU of computer.

The microprocessor communicates and operates in the binary number ‘0’ and ‘1’ called beta.

Each microprocessor has a fixed set of instructions in the form of binary patterns called a machine language.

The binary instructions are given abbreviated names,called mnemonics (opcode), which form the assembly language for a given microprocessor.

Definition:

 A microprocessor is a multipurpose,programmable,clock driven,register-based electronic device that reads binary instructions from a storage device called memory , accepts binary data as input and processes data according to those instructions and provides results as output.

Bus :- Bus is a group of wires which carry the information regarding the address or data or control signal. So, bus is mainly of 3 types :

1)  Address Bus : Address Bus is unidirectional. It carries the address of memory or input/output devices.

Address is always provided by the microprocessor to the peripheral devices.

When we want to fetch the content from any memory location or any other device , then the address travels through the address bus.

2) Data Bus : Data Bus is bidirectional because  and memory write operation , then the flow of data is from microprocessor to memory . Similarly , we explained in I/O devices.So,it is bidirectional.

3) Control Bus : Control Bus is also bidirectional . The control bus is comprised of various single lines that carry synchronization signal . The microprocessor uses such lines to provide timing signals : Read , Write , Ready.

The 8085 Microprocessor

The features of 8085 includes:

  1. It is an 8-bit microprocessor i.e. it can accept,process or provide 8-bit data simultaneously.
  2. It operates on a single +5V power supply.
  3. It has an chip clock generator.it divides oscillator frequency by 2 and generates clock

signal ,which can be used for synchronizing external devices.

  1. It can operate with 3MHz clock frequency.The 8085 A version can operate at the maximum frequency of 5MHz.
  2. It has 16 adress times, hence it can access (2^16) 64 kbytes of memory.
  3. It provides 8 bit I/O addresses to access (28) 256 I/O ports.
  4. 6600 transistors are used in fabrication and it was designed using NMDS technology.
  5. The lower 8 bit address bus (Ao-Ay)and data bus (Do-Dy) are multiplied to reduce the number of external pins.
  6. It supports 74 instructions.
  7. The 8085 is a von-neumann memory architecture.
  8. It provides five hardware interrupt:

TRAP, RST7.5,RST6.5,  RST5.5, and INTR.

  1. It has serial I/O control which allows serial communication.

Registers:The 8085 registers are classified as:

1. General Purpose Registers

B,C,D,E,H and L are 8-bit general purpose registers can be used as a separate 8-bit

separate 8-bit registers or as 16-bit register pairs BC,DE and HL.

HL pairs also function as a data pointer or memory pointer.

To store and read data from these registers bus access is not required,it is an internal

operation.The efficient programmer refers to use those registers than the memory locations

which require bus access and hence more time to perform the operation.

2. Temporary Registers

(a) Temporary Data Register 

The ALU has two inputs.One input is supplied by the accumulator and other from temporary

data register.The programmer cannot access this temporary data register.However,it is

internally used for execution of most of the arithmetic and logical instructions.

(b) W and Z Registers

W and Z are temporary registers.These registers used to hold 8-bit data during execution

of some instructions.These registers are not available for programmers,since 8085 uses them internally.

3. Special Purpose Registers

(a) Register A (Accumulator)

It is an 8-bit register.It is extensively used in arithmetic,logic,load and store operations,as

well as, I/O operations.Most of the times the result of arithmetic and logical operations is

stored in the RegisterA.Hence it is also identified as accumulator.

(b) FLAG Register

It is an 8-bit register,in which has information about the 5 flags:S (Sign Flag), Z (Zero Flag),

AC (Auxiliary Carry Flag),P(Parity Flag) and CY (Carry Flag) as shown below;

S-Sign Flag:After the execution of arithmetic or logical operations,if bit D9 of the result is 1,

the sign flag is set.In a given byte if D9 is 1,the number will be viewed as a negative number.

If D1 is 0,the number will be considered as a positive number.

Z-Zero Flag:The zero flag sets if the result of operation in ALU is zero and resets if results

is non-zero.

AC-Auxiliary Carry Flag:This flag is set if in an arithmetic operation,when a carry is

generated by digit D3 and based onto digit D4.This flag is used only internally for BCD

(binary-coded decimal) operations and is not available for the programmer to change the

sequences of a program with a jump instruction.

CY-Carry Flag:If an arithmetic operation results in a carry,the carry flag is set,otherwise it is

reset.The carry flag also serves as a borrow flag for subtraction.

arithmetic or logical operation if the result has an even number of ones,i.e., even parity,the

flag is set.If the party of odd,the flag is reset.

(c) Instruction Register : In a typical processor,operation first fetches the opcode of

instructions from memory.The processor stores this opcode in a register called the

instruction register.This opcode is further sent to the instruction decoder.

4.Sixteen Bit Registers 

(a) Program Counter (PC):Program is a sequence of instructions.Microprocessor fetches

there instructions from the memory and executes them sequentially.The program counter

stores the address of the next introduction to be fetched.For one byte introduction it

increments program counter by one,for 2byte it increments PC by two and for 3 byte it

increments PC by two and for 3byte it increments PC by three.

(b) Stack Pointer (SP):The stack is a reserved area of the memory in the RAM where

temporary  information may be stored.A 16 bit stack pointer is used to hold the address of

the most recent stack entry.

5.Arithmetic Logic Unit (ALU)

The 8085’s ALU performs arithmetic and logical functions on 8-bit data.The arithmetic

operations such as addition and subtraction.The logic unit performs logical operations such

as complement,AND,OR and PX-OR as well as rotate and clear.

Instruction Decoder

The instruction decoder decoder decodes instruction and accordingly gives the timing and

control signals.This information goes to timing and control circuitory.

Address Buffer

This is an 8-bit unidirectional buffer.It is used to drive external high order address bus

(A15-A8).It is also used to tri-state the high order address bus under certain conditions

such as resets,hold,half and when address lines are not is use.

Address/Data Buffer

This is an 8-bit bi-directional buffer.It is used to drive multiplexed address/data bus i.e., low

order address bus (A9-A10) and data bus (D1-D0).It is also used to tri-state the multiplexed

address/data bus under certain conditions such as reset,hold,halt and when the bus is not in

use.

Incrementer/Decrementer Address Lakh

This is used to increment or decrement the contents of program counter or stack pointer.

Interrupt Control

The processor fetches,decodes and executes instruction in a sequence.Sometimes it

necessary to have processor the automatically executes one of a collection of special

routines whenever special condition exists within a program or microcomputer system.The

occurence of this special condition is referred as interrupt.The interrupt control block has five

interrupt inputs RST 5.5,RST 6.5,RST 7.5.TRAP and INTR and one acknowledge signal

INTA.

Serial I/O Control

In serial communication one bit is transferred at a time over a single line.The

8085’s serial I/O control provides two lines,SOD and SID for serial

communication.The serial output data (SOD) line is used to send data serially

and serial input data (SID) line is used to receive data serially.

Timing and Control Circuitry

The control circuitry in the processor 8085 is responsible for all the operations.Hence the

operations in 8085 are synchronised with the help of clock signal.Along with the help of clock

signal.Along with the control of fetching and decoding operations and generating

appropriate signals for instructions execution,control circuitry also generates signal

required to interface external devices to the processor 8085.

Pin definitions of 8085

The signals of 8085 can be classified into seven groups according to functions.

(a)Power supply and frequency signals.

(b)Data bus and Address bus

(c)Control bus

(d)Interrupt signals

(e)Serial I/O signals

(f)DMA Signals

(g)Reset signals

(a)Power supply and frequency signals

(i)Vcc-It requires a single +5V power supply.

(ii)Vss-Ground reference

(iii)X1 and X2-A tuned ckt like LC,RC or crystal is connected at these two pins.The internal

clock generator divides oscillator frequency by 2,therefore to operate a system at 3MHz,the

crystal of tuned circuit must have a frequency of 6MHz.

(iv)CLK OUT-This signal is used as a system clock per other devices,its frequency is half the

oscillator frequency.

(b)Data bus and Address bus

AD0 to AD7-The 8bit data bus (D0-D7) is multiplexed with the lower half (A0-A7) of the 16bit

address bus.During first part of the machine cycle (T1),lower 8bits of memory address as I/O

address appear on the bus.During remaining  part of the machine cycle (T2 and T3) these

lines are used as bi-directional data bus.

A8 to A15-The upper half of the 16bit address appears on the address buses A8 to A15.

These lines are exclusively used for the higher order of address lines.

(c)Control and Status Signal

ALE (Address Latch Enable):The latching of lower half of an address bus is done by using

external latch and ALE signal from 8085.

RD and WR:A low on RD indicates that the data must be read from selected memory

location or I/O part via data bus.A low on WR indicates that the data must be written into the

Selected memory location or I/O part via data bus.

IO/M,S0 and S1:IO/M indicates whether I/O operation is being carried out.

            IO/M = 1     I/O related operation

                   = 0     memory related operation

S1 and S0 indicate the type of machine cycle in programs.

READY:It is used by the microprocessor to sense whether a peripheral is ready or not for

data transfer.If not,the processor waits.

(d)Interrupt signals

The 8085 has five hardware interrupt signals:RST 5.5,RST 6.5,RST 7.5,TRAP and INTR.

The microprocessor recognizes interrupt requests on these lines at the end of the current

instruction execution.

The INTA (Interrupt Acknowledge) signal is used to indicate that the processor has

acknowledged an INTR interrupt.

(e)Serial I/O signals

SID(Serial IP Data):This input signal is used to accept serial data bit by bit from external

device.

SOD(Serial OP Data):This is an output signal which enables the transmission of

serial data bit by bit to the external device.

(f)DMA Signals

HOLD:This signal indicates that another master is requesting for the use of address bus,

data bus and control bus.

HLDA:This active high signal is used to acknowledge HOLD requests.

(g)Reset signals

Reset IN:A low on this pin sets the PC to zero,resets the interrupt enable and HLDA

flip-flops,translate the data bus,address bus and control bus and affects the contents

of processed,internal registers randomly.

Reset OUT:The active high signal indicates that the process is being reset.

2.Reset Circuit

On reset,the PC set to ooooH which causes the 8085 to execute the first instruction from

oooH.For reset operation reset signal must be held low for at least 3 clock cycles.The power

on reset circuit can be used to ensure execution of first instruction from address ooooH.

Upon power up or key press the RESET IN goes low and slowly rises to +5V,providing

sufficient time for the processor to reset the system.The diode is connected to discharge the

capacitor immediately when power supply is switched OFF.

Demultiplexing AD7-AD0

The AD7-AD0 lines are multiplexed and the lower half of address (A7-A0) is available only

during T1 of the machine cycle.This lower half of address is also necessary during T2 and

T3 of of machine cycle to access specific location in memory or I/O port.This mean lower

half of address bus must be latched in T1 of the machine cycle,so that it is available

throughout the machine cycle.

The latching of lower half of an address is done using external latch and ALE signal from

8085.

Generation of Control Signals

The 8085 microprocessor provides RD and WR signals to initiate read or write cycle.Because there signals are used both for reading/writing memory and for reading/

writing an input device,it is necessary to generate separate read and write signals for

memory and I/O devices.

Using IO/M signal along with RD and WR,it is possible to generate separate four control

signals.

MEMR (Memory Read):To read from memory

MEMW(Memory Write):To read data in memory

IOR(I/O Read):To read data from I/O devices.

IOW(I/O Write):To write data in I/O devices.
