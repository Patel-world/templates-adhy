Into mapping -
If f : A → B be a mapping such that at least one element of B is
not f image of any element of A, then f is called Into mapping or A

Many One mapping -
A mapping f from A to B is said to be many one if two or more
than two elements of A have the same image of an element of B.

1. Reflexive Relation -
If R be a relation in a set A i.e. R is a subset of A × A then R is
called a reflexive relation if (a, a)←R, ∀ a←A
Hence R is reflexive if (a, a) ∈ R, ∀ a←A.
R is not reflexive if there is at least one element a←A, such
that (a, a) ∉ R.
Example -
If A = {1, 2, 3} then relation R = {(1, 1), (1, 3), (2, 2), (3, 3), (4,
4), (3, 4)} in A is reflexive, since ∀ a ∈ A ⇒ (a, a) ∈ R
If
∀ a ∈ A, (a, a) ∉ R then R is called irreflexive relation.
2. Symmetric -
(a, b) ∈ R ⇒ (b, a) ∈ R for all a,b ∈ A
3. Transitive -
If (a, b) ∈ R and (b, c) ∈ R ⇒ (a, c) ∈ R, for all a, b, c←A

Partial Order Relation -
A relation R on a set A is called partial order if it is reflexive,
antisymmetric and transitive, i.e.
1. (a, a) ∈ R for all a ∈ A
---Reflexive
2. (a, b) ∈ R and (b, a) ∈ R ⇒ a = b ----Antisymmetric
3. (a, b)
∈ R and (b, c) ∈ R ⇒ (a, c) ∈ R , -----Transitive
A set A together with a partial order R is called a partial order set
and is denoted by (A, R)
Example -
Show ≥ relation is a partial ordering on z, the set of integers.
Solution -
Reflexive - since a ≥ a ∀ integer a so greater therefore equal to is
reflexive
Anti - symmetric - since a ≥ b and b ≥ a ⇒ a = b
Therefore ≥ is anti - symmetric.
Transitive - since a ≥ b and b ≥ c ⇒ a ≥ c
Therefore ≥ is transitive
Hence ≥ is a partial order on z.
